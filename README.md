# statink_player_scanner
Stat.inkの戦績を順に確認し、過去対戦したプレイヤーを探す  

# 必要なもの
- stat.inkアカウント（ログインは不要）
- 検索したいユーザ名  

# ざっくりとした処理  
- スクレイピングでプレイヤー名を抜きだす
- 探したいプレイヤー名に部分一致する場合は名前を確認し、目的の名前でなければ無視リストに追加
- 「前のバトル」ボタンからURLを探して過去の試合を総当たり
